FRONTEND Y BACKEND

Hecho por: Violeta López Sánchez

Frontend

﻿﻿Frontend: Es la parte de una aplicación que los usuarios ven e interactúan directamente. Incluye el diseño, los botones, los menús y cualquier elemento visual. Se desarrolla con tecnologías como HTML, CSS y JavaScript, junto con frameworks como React, Angular o Vue.js.

BACKEND

Backend: Es la parte que funciona "detrás de las cámaras" y maneja la lógica, las bases de datos y la comunicación con el servidor. Se encarga de procesar datos, gestionar autenticaciones y ejecutar la lógica de la aplicación. Se desarrolla con lenguajes como Python (Django, Flask), JavaScript (Node.js), PHP, Ruby, Java o C.
